Getting Started

Add new pictures


  • Put some JPGs of the items (e.g. movies) into the DatabaseToGo Directory
  • Give them meaningful, clean names.
  • You may want to give them a standard height (e.g. 268 pixels) - use any imaging program for that task

Find new pictures


  • Start !db-to-go.exe
  • Enter '?' (one question mark) into the upper right search area (this will search all files without attributes).

Add new attributes to the pool


  • Click into the right area below the file name - the Attributes Window shows up (yet empty).
  • Enter any attribute name (e.g. 'Genre') into left upper field and press [Enter]
  • Enter any attribute value (e.g. 'Science Fiction') into the middle top field and press [Enter].
  • Enter another attribute name (e.g. 'Year') into left upper field and press [Enter]
  • Enter another attribute value (e.g. '1984') into the middle top field and press [Enter].
  • You should now have 2 attributes witch 1 value each.

Assign attributes to pictures


  • Select the first attribute name ('Genre')
  • Double click on the value('Science Fiction')
  • Select the second attribute name ('Year')
  • Double click on the value('1984')
