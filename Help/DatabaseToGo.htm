Introduction

DatabaseToGo is a free, open-source utility for windows. With it, you can:

  • Organize your collections (movies, any kind of inventory) in the form of pictures
  • Add any kind of attributes to the pictures (not only the usual JPG-attributes)
  • Create different structures (tree views) for the same data
  • Do real-time search for file names or attributes
  • Keep the data synchronized between many devices (hard disks, USB sticks etc.)
  • View the data directly from an USB stick without installing anything

How it works

  • The 'database' is actually a simple folder containing JPGs and the viewer.
  • The meta data (your attributes) is stored inside the JPG-files
  • View the data directly from an USB stick without installing anything
